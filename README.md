# Creating an AWS Homelab

## Objective

To build and document a functional cloud homelab on AWS, demonstrating the ability to provision, configure, and manage a Linux-based virtual server from scratch.

### Skills Learned

- Cloud Infrastructure Provisioning
- Network Security Configuration
- Remote Server Administration
- Web Server Deployment
- Linux System Administration
- Cloud Networking Fundamentals

### Tools Used

- AWS EC2
- AWS Management Console
- AWS VPC
- Ubuntu Server 24.04 LTS
- Windows Command-Line

## Steps
1. Create AWS Account
2. Launch EC2 Instance
3. Configure Key Pair
4. Configure Network Settings
5. Launch and Connect via SSH
6. Launch and Connect via EC2 Instance Connect (Public IP)
